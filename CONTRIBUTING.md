# Contributing to Spanish Phrases for Travelers

Thanks for your interest! Here's how to contribute:

## Reporting Issues
- Use GitHub Issues for bugs or suggestions.
- Provide details: description, steps to reproduce, screenshots.

## Pull Requests
- Fix typos in phrases/pronunciations.
- Add new phrases or sections.
- Improve UI/UX.
- Ensure code is mobile-friendly.

Steps:
1. Fork and clone.
2. Install no dependencies needed.
3. Test locally.
4. Commit with clear messages.
5. PR against main.
