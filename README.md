# Spanish Phrases for Travelers: Mexico & Spain

An interactive HTML guide with 1,400+ Spanish phrases, audio pronunciation, cultural tips, and collapsible sections for traveling in Mexico and Spain. Built with HTML, JS, and CSS for offline use on mobile or desktop.

![Screenshot of the guide](screenshot.png) <!-- Add a screenshot image to the repo and reference it here -->

## Demo
View the live guide here: https://v1738.github.io/spanish-phrases-for-travelers-mexico-spain/

## Features
- **Audio Pronunciation**: Click "Listen" to hear phrases spoken (using browser speech synthesis).
- **Dark Mode**: Toggle for better readability.
- **Mobile Optimized**: Responsive design for phones and tablets.
- **Cultural Tips**: Etiquette for Mexico and Spain.
- **Categories**: Organized into sections like greetings, transportation, food, emergencies, etc.
- **Offline Ready**: Download and use without internet.

## How to Use
1. Open `index.html` in any modern browser.
2. Browse sections using collapsible details.
3. Search for keywords to filter phrases.
4. Click "Listen" for audio.

For development:
- Edit HTML/CSS/JS directly.
- No build tools needed.
